Custom themes

Custom button themes can be generated using the Vaadin Valo theme engine (i.e. "vaadin-theme").

  1. Install vaadin-theme globally as a service (requires Node.js):

    $ npm install -g vaadin-theme

  2. Create a Sass file, e.g. "custom-theme.scss" which describes the new theme, using the Valo theme API:

    $v-theme-name: "my-theme";
$v-included-components: button;
$v-font-family: Helvetica Neue, sans-serif;
$v-font-weight: 500;
$v-border: 0;
$v-border-radius: 0;
$v-background-color: #00b4f0;
$v-bevel: false;
$v-shadow: false;
$v-gradient: false;
$v-unit-size: 40px;

@import "vaadin-theme";

  3. Use the command line to compile the file into a new theme:

    $ vaadin-theme custom-theme.scss

    The output is generated into a file named "custom-theme.css" (i.e. the same file name as the input but a different file type), in the same folder as the input. You can also specify an explicit output path as a second argument for the command:

    $ vaadin-theme custom-theme.scss path/to/output.css

  4. Include the new file in your HTML page as any other CSS stylesheet:

    <link rel="stylesheet" href="path/to/custom-theme.css">

  5. Specify the theme attribute for the buttons you wish to use the custom theme. The attribute value should match the $v-theme-name Sass variable value:

    <v-button theme="my-theme">Custom theme<v-button>
